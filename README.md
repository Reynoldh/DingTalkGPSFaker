# iOS 逆向实战 - 钉钉签到远程“打卡”

> 更新：此版本功能较简单，使用起来也不是特别方便，目前正在开发增强版，可支持地图选位置伪装 GPS 定位，同时支持 WIFI 打卡。现在已经实现 WIFI 打卡功能，具体请参考 [DingTalkAssistant](https://github.com/buginux/DingTalkAssistant)。

本 Tweak 可以模拟钉钉的 GPS 定位，伪装位置，实现在家也能打卡的功能。
